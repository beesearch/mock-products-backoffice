#  Products backoffice 
#### Mock data for bee-backend

### Sqlite3
```
// Install Sqlite
brew install Sqlite3 	//MAC OS X
apt-get install sqlite3 //Ubuntu

// Use the database
sqlite3 products.db
// Reset the database
sqlite3 products.db < products.sql
```

### Application
```
npm install
npm install sqlite3
```

### Start the server
```
node server.js
```
